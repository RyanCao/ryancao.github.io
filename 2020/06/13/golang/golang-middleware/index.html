一个 Api 应该不能随意调用

  1. 玩家登陆 带username/passwd验证。服务器返回Token，可以根据token设置权限请求信息
  2. 玩家获取到Token后，直接访问Api资源
  3. 服务器获取请求后解析T验证oken信息，Token验证通过则返回对应的资源，否则不返回资源信息
