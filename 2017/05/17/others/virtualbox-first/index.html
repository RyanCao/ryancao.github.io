虚拟机操作系统 centos7

  1. 安装增强功能

菜单位置

设备 -> 安装增强功能

Vbox挂载了一个光盘cdrom的硬件(只读)

如果失败，则需要查看Log

一般需要安装编译环境

kernel

确认当前使用的kernel版本号: uname -r

查找当前系统安装的所有kernel: [rpm -qa|grep kernel]

安装 kernel-devel

想要安装VirtualTool 需要 kernel-devel 版本号 和 kernel 版本号一致

由于版本号不一致：
决定升级 yum

再次查看当前系统安装的所有kernel: [rpm -qa|grep kernel]

安装 kernel-devel


或者 安装 dkms

切换内核693登录,再次执行安装程序

然后继续 执行
