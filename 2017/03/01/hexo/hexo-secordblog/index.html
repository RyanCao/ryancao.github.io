Hexo配置

hexo里面有两个常用到的配置文件 ${hexo} 项目根文件夹

  • 整个博客的配置文件 ${hexo}_config.yml
  • 主题的配置文件 ${hexo}\themes\xxx_config.yml

hexo3.0的默认主题是landscape，就是 ${hexo}\themes\landscape_config.yml

替换主题

  • 找到你需要的喜欢的主题，下载下来，我用的是Next
  • 把主题文件放到${hexo}\themes 下
  • 修改博客配置文件 ${hexo}_config.yml 中的theme名字改为自己下载的主题的文件夹名即可。
